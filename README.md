# SETLIST NOTE

シンプルなセットリスト作成アプリの初期版です。

## 起動方法

```bash
npm install
npm run dev
```

ブラウザで `http://localhost:3000` を開いてください。

## 現在できること

- イベント・アーティスト情報の入力
- 楽曲の追加、削除、並べ替え
- 曲ごとの演奏時間、音響、照明、メモ入力
- 全体要望・特記事項の入力
- localStorageへの自動保存
- A4横向きの印刷・PDF保存
- スマホ表示対応
